# 🎯 DSA & Coding

> **Phase:** 8 | **Duration:** ~2 weeks | **Priority:** High
> **Related:** [[02 - SQL Coding Problems]], [[01 - Behavioral Leadership]]

---

## Checklist

### Arrays & Strings
- [ ] Two-pointer technique: sorted array problems, palindromes, container with most water
- [ ] Sliding window: fixed-size and variable-size windows, max subarray
- [ ] Prefix sums: range sum queries, subarray sum equals K
- [ ] Kadane's algorithm: maximum subarray sum
- [ ] String manipulation: anagrams, substring search, string matching
- [ ] Practice: 10-15 problems (LeetCode Medium)

### Hash Maps & Sets
- [ ] Frequency counting: character/word frequency, top-K frequent
- [ ] Two Sum pattern: complement lookup
- [ ] Grouping: group anagrams, group by property
- [ ] Sliding window + hash map: longest substring without repeating chars
- [ ] Design: LRU Cache (LinkedHashMap)
- [ ] Practice: 8-10 problems

### Trees & Graphs
- [ ] BFS: level-order traversal, shortest path in unweighted graph
- [ ] DFS: pre/in/post-order traversal, path finding, cycle detection
- [ ] Topological sort: DAG ordering (relevant to Airflow DAGs!)
- [ ] Binary search tree: insert, delete, search, validate BST
- [ ] Graph representations: adjacency list vs matrix
- [ ] Union-Find: connected components, cycle detection in undirected graph
- [ ] Dijkstra's: shortest path in weighted graph (awareness)
- [ ] Practice: 10-12 problems

### Dynamic Programming
- [ ] Memoization (top-down) vs tabulation (bottom-up)
- [ ] Key patterns: 0/1 knapsack, unbounded knapsack, longest common subsequence
- [ ] Fibonacci-type: climbing stairs, house robber, coin change
- [ ] Grid problems: unique paths, minimum path sum
- [ ] String DP: edit distance, longest palindromic subsequence
- [ ] Practice: 8-10 problems (focus on patterns, not memorization)

### Sorting & Searching
- [ ] Binary search: on sorted arrays, search space reduction, bisect_left/bisect_right
- [ ] Merge sort: divide and conquer, stable, O(n log n)
- [ ] Quick sort: partition, average O(n log n), worst O(n²)
- [ ] Heap: top-K elements, K-th largest, merge K sorted lists
- [ ] Practice: 5-8 problems

### Data Engineering Coding
- [ ] Parse and aggregate log files: extract patterns, count occurrences
- [ ] Deduplicate events: by key + timestamp, keep latest
- [ ] Sessionization: group events into sessions by time gap
- [ ] Implement mini ETL: read CSV → transform → write Parquet (Pandas/Spark)
- [ ] Data reconciliation: compare two datasets, find mismatches
- [ ] Stream processing mock: process events in order, maintain state

### Spark/Pandas Coding
- [ ] GroupBy aggregations: multiple aggs, custom agg functions
- [ ] Pivot/unpivot: wide ↔ long format transformations
- [ ] Time-series resampling: resample, forward-fill, rolling windows
- [ ] Join patterns: left join with null handling, self-join for comparisons
- [ ] Window functions: rank within groups, running totals
- [ ] Data cleaning: handle nulls, type conversions, dedup
