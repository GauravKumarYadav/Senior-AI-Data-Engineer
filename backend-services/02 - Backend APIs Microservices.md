## Checklist

### API Patterns
- [ ] REST: resource-oriented, HTTP methods, status codes
- [ ] gRPC: Protocol Buffers, streaming RPCs, ML model serving
- [ ] GraphQL: client-defined queries (awareness level)
- [ ] WebSocket: bidirectional real-time communication

### Microservices
- [ ] Service decomposition: bounded contexts, single responsibility
- [ ] Communication: synchronous (REST/gRPC) vs async (message queue)
- [ ] Message queues: RabbitMQ, AWS SQS — decoupling services
- [ ] API Gateway: Kong, AWS API Gateway — routing, auth, rate limiting
- [ ] Service discovery: DNS-based, Consul, K8s services
- [ ] Circuit breaker: fail fast, prevent cascade failures (resilience4j pattern)
