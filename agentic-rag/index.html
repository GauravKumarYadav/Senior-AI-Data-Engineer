What This App Does

This course is built around a real, production-grade Agentic RAG chatbot — a system where an AI doesn't just generate answers, it decides how to find them. It routes questions to specialized agents, searches multiple document collections, reranks results, manages memory, and cites its sources — all orchestrated by a LangGraph state machine.

🧠
Multi-Agent Architecture
4 specialized agents (Query, Retrieval, Tool, Synthesis) coordinated by a LangGraph orchestrator that routes based on intent.
🔍
Hybrid Search
BM25 keyword search + vector semantic search fused with Reciprocal Rank Fusion, then reranked by a cross-encoder.
🏢
Client Isolation
Per-client document collections with physical separation in ChromaDB. Global documents available to all.
💬
Smart Memory
Redis-backed session buffer with auto-summarization when conversations get long. Circuit breaker for resilience.
📊
Full Observability
Every agent step traced in LangSmith. Cost tracking, structured JSON logging, and correlation IDs.
🔌
Multi-Provider LLM
Supports LM Studio, Ollama, Groq, OpenAI, or any OpenAI-compatible endpoint. Switch with one config change.
