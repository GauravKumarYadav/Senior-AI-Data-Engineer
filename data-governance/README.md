# Data Governance Interview Mastery

Interactive course covering data governance for data engineering interviews — RBAC vs ABAC, PII handling, GDPR compliance, data catalogs, retention policies, and audit strategies.

## What's Inside

- **6 Modules** with interactive quizzes and visual diagrams
- Covers roles (owner/steward/custodian), masking, tokenization, row/column-level security, and more
- Scenario-based quizzes testing real compliance decisions

## Modules

1. What Is Data Governance?
2. Roles, Ownership & Catalogs
3. Access Control — RBAC & ABAC
4. PII, Masking & Tokenization
5. Compliance, GDPR & Retention
6. The Interview Gauntlet
